## Goal

`/generate` と `/generate?feature=...` の画面骨格を Lightchain と同じ4カテゴリ・機能一覧・選択中入力パネルへ作り替える。

## Scope

- `おすすめ`, `企画デザインツール`, `AIフィッティング`, `グラフィックツール` を主導線にする。
- 旧カテゴリ、運用ステータス、Runway詳細、UsageStats は第一導線から外し、必要時のみ折りたたみへ退避する。
- Heavy Chain独自機能は4カテゴリに自然に統合する。

## Done Evidence

- Desktop `/generate` screenshot and DOM show `HEAVY CHAIN AI` plus exactly the 4 main category labels.
- Desktop selected feature screen shows left category feature list, central input/material/generate panel, and result panel.
- Mobile `/generate` screenshot does not overlap, clip, or show old top-level noise first.
- `npm run typecheck`, `npm run lint -- --max-warnings=0`, `npm run build`, `git diff --check` pass.
