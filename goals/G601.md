# G601 Fresh All-Feature Generation QA

/goal 全10主要生成機能の既存fresh proofを再監査し、不足・古い・弱い証跡だけを bounded fresh generation で補い、job/task/readback/storage/image files/visual scorecard で完了を示す。Runway workspace limit 等で必要分のfreshが不可能な場合は、受け入れ済みproof、再実行成功分、失敗分、exact blocker、再開条件を分けて記録する。課金、購入、支払い、本人確認、OTP/CAPTCHA、秘密入力、外部公開、旧localhost OAuth経路は禁止。検証は marker-scoped jobs/images/usage rows と cleanup proof を使う。

## Scope

Features: `campaign-image`, `product-shots`, `model-matrix`, `design-gacha`, `scene-coordinate`, `multilingual-banner`, `remove-bg`, `colorize`, `upscale`, `variations`.

## Done Evidence

- Run manifest and per-feature prompts.
- Runway task IDs or exact blocker.
- Worker/import DB/Storage readback.
- Downloaded images/contact sheet.
- Visual scorecard.
- Marker-scoped cleanup proof when rows were created.
