# G607 Release Gate Unification

/goal production monitor、launch-ops、mass-market QA、security/performance checks を、リリース前に実行できる一つの手順へ統合する。pass/fail contract、artifact path、停止条件、復旧手順を明確にする。課金、購入、支払い、本人確認、OTP/CAPTCHA、秘密入力、外部公開、破壊的cleanup、新しい有料外部監視サービスは禁止。

## Done Evidence

- Script or runbook update.
- Dry-run/read-only run proof.
- Pass/fail summary artifact.
- Static checks and Codex review if code changed.
