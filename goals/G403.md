# G403 Output And Generation QA

## Goal

生成成果物が Lightchain と同じ感覚で確認でき、Heavy Chain の生成結果がプロンプト意図に合うことを実画像・readback・視覚スコアで確認する。

## Scope

- Non-billing marker-scoped generation only.
- Approved Runway MCP / local worker flow only.
- Existing verified generated images may be reused only as historical comparison; current changed UI must still be newly recorded.

## Done Evidence

- At least one current non-billing generation or accepted recent generated proof is opened and visually scored.
- Gallery/History/Canvas continuation is verified by URL/DOM/screenshot/video.
- DB/Storage/readback and cleanup proof are captured when new marker-scoped rows/images are created.
- Visual scorecard covers prompt adherence, apparel fidelity, unwanted text/watermark/UI artifacts, composition, and commercial usefulness.

## Child Goal Text

`/goal Prove Heavy Chain generated outputs and continuation flows work in the Lightchain-like UI, verified by generated image inspection, DB/Storage/readback when new rows are created, Gallery/History/Canvas evidence, and a visual scorecard, while avoiding billing/payment/external publishing. If blocked, report exact provider/tool/auth/input blocker and saved proof.`
