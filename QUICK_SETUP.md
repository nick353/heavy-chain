# ⚡ クイックガイド: GEMINI_IMAGE_MODEL 設定

## 🎯 やること（1分で完了）

今開いているSupabaseの画面で、新しい環境変数を追加するだけです。

---

## 📝 コピー&ペースト用

### 環境変数名:
```
GEMINI_IMAGE_MODEL
```

### 値:
```
imagen-3.0-generate-001
```

---

## 🖱️ クリック手順

1. **「Add new secret」** ボタンをクリック
2. Name欄に `GEMINI_IMAGE_MODEL` を貼り付け
3. Value欄に `imagen-3.0-generate-001` を貼り付け
4. **「Save」** または **「Create」** をクリック

✅ 完了！

---

## 🧪 テスト

https://heavy-chain.zeabur.app にアクセス
→ ログイン
→ 画像生成
→ 商品説明を入力して「生成」

**画像が生成されれば成功です！** 🎉

## ❓ よくある質問

### Q: 大文字・小文字は重要？
**A**: はい！ `GEMINI_IMAGE_MODEL` と正確に入力してください（すべて大文字）

### Q: モデル名は？
**A**: `imagen-3.0-generate-001` （ハイフン、ピリオドも正確に）

### Q: 設定後すぐに使える？
**A**: はい！保存した瞬間から有効です

### Q: エラーが出たら？
**A**: ブラウザでF12キーを押してコンソールを開き、エラーメッセージを確認してください
