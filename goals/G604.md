# G604 Failure Recovery UX

/goal Runway制限、worker待ち、参照画像失敗、生成失敗、network/API失敗を、ユーザーが次に何をすればよいか分かるUIへ改善する。検証は forced/simulated failure、DOM/screenshot、copy review、retry/next-action確認、関連テストで行う。実課金、外部公開、秘密入力、破壊的cleanupは禁止。

## Done Evidence

- Failure matrix.
- Before/after screenshots or DOM.
- Tests/verifier updates.
- User-facing Japanese copy quality pass.
- Remaining external blockers if any.
